The Objective →

CREATE A PATTERN USING A CREATIVE CODING LANGUAGE,
PROCESSING.

Concept →

Portugese Tiles/ Azulejos

This project is inspired by the tin-glazed ceramic tile works which can be seen on the interior or exterior of many places in Portugal.

Azulejos designs are usually made of simple shapes that are rotated and repeated to create intricate patterns. Moreover, the overall feeling of the design is both organic and structured at the same time.

CREATE A FINAL COLORED-DESIGN COMPOSITION BEFORE MOVING INTO PROCESSING.

THE SHAPE →

First, draw one eighth of first shape using triangle(), vertex(), and bezierVertex() functions.

Rotate and flip the one eighth part into into a full rotation
using for(), rotate(), and scale() functions.

Create one eighth of the second shape using vertex() and bezierVertex() functions.

Repeat the rotating and flipping process to create the second shape.

THE PATTERN →

Repeat the shapes using the for() function to make the pattern.

INTERACTIVITY →

Allow the pattern to change its background color to blueGreen
by clicking the left mouse using draw() and if() functions.

Allow the pattern to change its background color to orange
by clicking the right mouse using draw() and else if() functions.
